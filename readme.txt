== Description ==
Got spammers? Add Recaptcha V3 in an instant! Works for any contact form. Uses Google Recaptcha V3. A step by step guide is included in the form so you won't get lost in the jungle of Google services.

== Installation ==
1. Upload the unpacked folder to the "/wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
